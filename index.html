My Time in SNHU

During my time here in SNHU, I have learned a lot about useful computer fields and other skills. However I have some which I really appreciate and they are going to be part of me throughout my life time.

Data structures and algorithm is one of the classes that I obtained a lot. I can comfortable work on existing algorithms and also I am able to create my own algorithms that can efficiently solve some specific problems. It’s the class which dragged me into loving java programming.

Learning data structures is the most important thing that I discovered, I have learned that they are like the basic or core things in a program. The number of lines of codes in a program entirely depends on one’s knowledge on data structures, the higher the broad the knowledge the fewer the lines of codes. Therefore struggled to learn data structures, and with the help of my instructors I can honestly say that I have gained a lot and I am capable of handling data structures in most of the computer languages like java and python.

With the help of the above class work, I can manage to create a computer program that can efficiently run in computers and solve specified problems. My main objective now is to work up and down to apply my skills so that I can create computer applications that can be usefully to my society.

Another usefully computer fields that I have learned and I am capable of applying their skills in real world situations is Software design and engineering. This is a field in a computer which controls the use of most of the protocols that govern structure of most programs. Each programming language comes with rules, further there are standard rules that govern the way programmers put they codes together, this is to help other developers to understand the code and also allow for group working in bigger projects.

Finally, I can say that I have learned a lot during my entire time in SNHU. I managed to learn and achieved my goals in most of the classes. I can work both as a front end developer or a back end developer.
